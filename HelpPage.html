GMMH Database

The main purpose of this database is to store data from 'Anesthesia and surgery induce age-dependent changes in behaviors and microbiota'.

  • Store data in Gut Microbiota - Mental Health research
  • Automatically visualize the data for users
  • Allow users to download data from database

How to visualize your data?

Provide the address of the data that you want to visualize

The data should be CSV file, the format that we can visualize automatically :)

The input CSV file needs to meet the following requirements: each column of the dataset is a sample, and each row is a feature. Just like the example on the left.

Note in particular that you need to provide the address of the data set rather than the CSV file itself.

About Data

  • Discription of Data

    All data are related to gut microbiota and mental health, such as species distribution in gut, or changes in mental behaviors (as shown by results in a maze) and brain IL-6 levels under different experimental conditions (ie given probiotics or not post-surgery). This is unpublished data from mgh lab (the format of unpublished data is csv) .
